# Task Scheduler Optimization System

## Overview

Task Scheduler Optimization System is a Python-based DSA project that optimizes task execution using sorting, priority queues (heap), and a greedy scheduling algorithm.

The system schedules tasks based on:

- Priority
- Deadline
- Execution Time
- Profit

It generates an optimized schedule while identifying late tasks and calculating overall performance.
